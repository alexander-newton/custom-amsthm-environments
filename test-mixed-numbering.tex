This document tests continuous numbering across BOTH custom environments
AND Quarto's built-in types.

Theorem]\protect\hypertarget{thm-builtin1}{}\label{thm-builtin1}

This is Quarto's built-in theorem. Should be 1.1.

\end{theorem}

\begin{problem}\label{prm-custom1}

Custom Problem environment. Should be 1.2 (continuing from theorem).

\end{problem}

\begin{definition}[]\protect\hypertarget{def-builtin1}{}\label{def-builtin1}

Built-in Definition. Should be 1.3 (continuing the sequence).

\end{definition}

\begin{claim}[Important Claim]\label{clm-custom1}

Custom Claim environment. Should be 1.4 (still continuing).

\end{claim}

\begin{lemma}[]\protect\hypertarget{lem-builtin1}{}\label{lem-builtin1}

Built-in Lemma. Should be 1.5.

\end{lemma}

The goal is: Theorem 1.1, Problem 1.2, Definition 1.3, Claim 1.4, Lemma
1.5 - all sharing the same counter!
